# 📚 图书管理系统

一个现代化的前后分离图书管理应用，使用 React、Spring Boot 和 H2 数据库构建。

## 项目特性

- ✨ **现代化UI** - 响应式设计，支持移动端
- 🔍 **智能搜索** - 按标题、作者搜索书籍
- 📊 **书籍管理** - CRUD 操作完整支持
- 📤 **借还管理** - 追踪书籍借出/还回状态
- 🐳 **Docker 支持** - 一键部署
- ⚙️ **CI/CD 流程** - GitHub Actions 自动化测试和部署

## 技术栈

### 前端
- **React 18** - UI 框架
- **Axios** - HTTP 客户端
- **Vite** - 前端构建工具
- **现代 CSS** - 响应式设计

### 后端
- **Spring Boot 3.2** - Java 框架
- **Spring Data JPA** - 数据访问
- **H2 数据库** - 轻量级关系数据库
- **Maven** - 项目管理

- Node.js 18+
- Docker & Docker Compose (可选)
